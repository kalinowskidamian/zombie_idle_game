# Zombie Idle Game

Prototyp gry Unity 2D (Android): idle city builder na cmentarzu z motywem zombie.

## Jak uruchomić lokalnie

1. Otwórz repozytorium w Unity Hub jako projekt Unity 2D.
2. Uruchom projekt w edytorze (Play) do szybkich testów.
3. Dla Androida: przełącz platformę na **Android** i zbuduj APK/AAB z poziomu Unity.

## Struktura folderów

Planowana podstawowa struktura:

- `Assets/Scripts/` – gameplay i logika gry.
- `Assets/Art/` – grafiki 2D, sprite’y, UI.
- `Assets/Audio/` – efekty dźwiękowe i muzyka.
- `Assets/Prefabs/` – prefaby obiektów.
- `Assets/Scenes/` – sceny Unity.
